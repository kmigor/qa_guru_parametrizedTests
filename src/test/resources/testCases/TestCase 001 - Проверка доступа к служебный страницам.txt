Тест-кейс:      Проверка доступа к служебный страницам.
ID тест-кейса:  001.

===============================================

Описание:       Проверка доступа к служебным страницам с главной страницы википедии (Страницы: "Содержание", "Избранные статьи", "Текущие события").

===============================================

Шаги:
1) Открыть страницу сайта (https://ru.wikipedia.org);
2) Перейти на служебную страницу из панели навигации слева;

===============================================

Ожидаемый результат:
1) Открылась служебная страница;
2) Заголовок страницы должен совпадать с названием из панели навигации.

===============================================

Фактический результат:
1) Открылось каждая служебная страница.;
2) Для каждой служебной страницы заголовок совпадает с названием старницы из панели навигации.

===============================================

Статус: Пройден
